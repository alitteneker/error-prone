After running analyze-testresults.sh in the above directory, this folder will contain two cdv files. Their contents are as follows:
  (1) stack.csv
       (a) stack id
       (b) variable number of package/class.method
  (2) stats.csv
       (a) stack id
       (b) passes
       (c) fails
       (d) Tarantula core
       (e) SBI score
       (f) Jaccard score
       (g) Ochiai score
       (h) Cumulative cyclomatic complexity
